How to use the worksheets

Everything happens on the two-line display — no pop-ups. Open a worksheet, then step through its prompts:

  • Enter a value: type digits, press ENTER.
  • Move between prompts: ↑ / ↓.
  • Compute the shown value: CPT.
  • Change a setting (e.g. ACT/360, END/BGN, method): 2ND SET.
  • Exit a worksheet: 2ND QUIT.

TVM: type a value then press N, I/Y, PV, PMT or FV to store; CPT then a key to solve. Money out is negative. Default P/Y = C/Y = 1.

2ND: press 2ND then a key for the small label above it — e.g. cash-flow keys, 2ND BOND (over 9), 2ND DEPR (over 4), 2ND P/Y (over I/Y), 2ND FORMAT (over .) to set decimals.

Dates are entered as MM.DDYYYY (e.g. 12.312024 → 12-31-2024).

Keyboard: digits, + - * / =, Enter, ↑/↓, Backspace (→), Esc (clear), c (CPT), Shift (2ND), q (quit worksheet), n (±).
